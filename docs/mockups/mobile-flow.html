Logique d'enchainement

Le Guild Home est le hub de retour. Il montre ce qui est fini, ce qui rapporte maintenant, et la prochaine amelioration evidente.

Regle UX

Chaque ecran doit avoir une action principale claire. Le joueur ne doit jamais chercher longtemps comment relancer une expedition.

Navigation MVP

  • Guild pour collecter et comprendre l'etat actuel.
  • Quests pour relancer le timer idle.
  • Heroes et Loot pour optimiser.
  • Upgrades pour creer un objectif long terme.

Ce que cette maquette valide

  • La boucle retour -> recompense -> optimisation -> nouvelle quete.
  • La place discrete de la monetisation.
  • Les ecrans indispensables avant le premier build Android.
