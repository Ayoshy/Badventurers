# Android Source Sheets

This folder keeps source/contact sheets that should not be packaged as runtime Android drawable resources.

Runtime drawable resources are split by category under `app/src/main/res-*` roots and wired through `app/build.gradle.kts`.

- `loot_icons_sheet.png`: 5x2 loot icon concept sheet.
- `quest_cards_sheet.png`: staging quest card concept sheet.
- `resource_ui_icons_sheet.png`: resource/status icon concept sheet.
- `starter_heroes_sheet.png`: starter hero portrait concept sheet.
