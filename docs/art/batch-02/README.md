# Batch 02: Extended Hero Portraits

Generated: 2026-06-22

This batch extends the starter hero portrait set from 4 to 8 class artworks while keeping the Batch 01 visual direction.

## Files

- `extended-heroes-sheet.png`: 2x2 concept sheet for Ninja, Hunter, Priest, and Necromancer.

## Android Slices

Generated Android-ready PNGs live in `app/src/main/res/drawable-nodpi/`:

- `hero_portrait_ninja.png`
- `hero_portrait_hunter.png`
- `hero_portrait_priest.png`
- `hero_portrait_necromancer.png`

Each portrait is sliced from the 1254x1254 source sheet into a 627x627 PNG to match the Batch 01 portrait dimensions.

## Direction

Match the existing polished 16-bit-inspired portraits: ornate parchment and brass frames, cozy tavern/guild backdrops, warm amber lighting, readable character silhouettes, expressive faces, and class-identifying props. No labels, text, watermarks, or modern elements.
