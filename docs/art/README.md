# Art References

This folder stores project-bound art references and generated concept targets for Badventurers.

## Files

- `badventurers-style-target-01.png`: first polished 16-bit pixel art direction target for the guild/tavern identity.
- `batch-01/`: prototype art pack for the first Android visual slice.
- `batch-02/`: extended hero portrait sheet for Ninja, Hunter, Priest, and Necromancer.
- `batch-03/`: new hero portrait batch for Paladin, Comptable, Jardinier, Chevalier de la mort, Chef Cuistot, Expert en démolition, Elémentaire de sel, and Troll Stupide.

## Usage

These images are reference targets, not final production assets. They guide visual direction, palette, polish level, and mobile readability before final app assets are produced.
