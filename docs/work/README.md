# Badventurers Workboard

The active task tracker is now `docs/work/tasks.json`, viewed and edited through the local board in `tools/task-board/`.

Run it from the repo root:

```powershell
node tools/task-board/server.mjs
```

Then open:

```text
http://127.0.0.1:4173
```

Codex can update the JSON directly or use the local API. The old `docs/product/kanban.md` remains historical product context.
